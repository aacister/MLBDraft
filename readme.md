
# MLBDraft (In progress) #

MLBDraft is used to simulate online fantasy baseball snake drafts.

Includes 2 projects:
 1. MlbDraft.Api
 2. MlbDraft.Client

## MlbDraft.Api ##
 * Developed utilizing Asp.net Core Web Api 2
 * SqLite data storage
 * JWT auth
 * Entity Framework
 * AutoMapper
 * NLog
