# MLBDraft #

Application is a hosting location for fantasy baseball leagues to host an online snake draft.

Includes 2 projects:
 1. MlbDraft.Api
 2. MlbDraft.Client

## MlbDraft.Api ##
 * Developed utilizing Asp.net Core Web Api 2
 * SqLite data storage
 * JWT authorization
 * Entity Framework
 * AutoMapper
 * NLog
